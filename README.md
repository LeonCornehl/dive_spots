# 🤿 Dive Spots

Analyzing dive sites around the world.

## Projects

### 🇲🇹 Malta
Exploratory data analysis of 233 dive sites across Malta, Gozo and Comino.

**Tools:** Python, Pandas, Folium, Plotly, geopy  
**Data:** [maltadives.com](https://maltadives.com/sites/en)  
**Notebook:** [malta/notebooks/malta_dive_sites.ipynb](malta/notebooks/malta_dive_sites.ipynb)

**Key findings:**
- Gozo has on average deeper and more challenging dive sites than Malta
- Wrecks tend to be rated higher than reefs
- Many top-rated sites are only accessible by boat
